Caesar Cipher Brute Force Script

A simple Python script created with the help of AI to brute-force all possible shifts of a Caesar cipher to help decrypt encoded messages. This script iterates through all 25 possible shifts and prints the decrypted text for each shift. The script is useful for educational purposes and demonstrates the concept of brute-force attacks in cryptography. 

# caesar_brute_force.py
cipher_text = "Wklv lv d whvw phvvdjh"
for shift in range(26):
    decrypted = ""
    for char in cipher_text:
        if char.isalpha():
            offset = 65 if char.isupper() else 97
            decrypted += chr((ord(char) - offset - shift) % 26 + offset)
        else:
            decrypted += char
    print(f"Shift {shift}: {decrypted}")
